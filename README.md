# Housing-Market-Project

This is a summer project that I am working on. The idea behind this project is that I found some data for a housing market that includes variables for each home. From these variables I will predict the price of each house. However, I want this to be more applicable for non-statisticians, the way that I will do that is by taking that predicted price and subtracting the actual price to determine whether the house is a good deal or not.
